Components library for Borland Delphi 7.

Includes:
  LsGroupBox - advanced TGroupBox component with colored rounded borders and some other funny things;
  LsOpenSndDlg - audio file selection dialog box with the ability to listen;
  LsColorButton - color selection button;
  LsFontPanel - font selection panel;
  LsAniLable - animated label;
  TheOnly - non-visual component to prevent multiple instances of the application from running.

These is not all the components, the rest will be published soon.
